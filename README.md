# MERN Notes App

A simple full-stack notes application built with the MERN stack.

## Tech Stack

Frontend
- React
- TailwindCSS
- Axios

Backend
- Node.js
- Express
- MongoDB
- Mongoose

## Features

- Create notes
- View notes
- Delete notes

## Run Locally

Backend

cd notes-backend
npm install
npm run dev

Frontend

cd notes-frontend
npm install
npm run dev
